规格参数
~~~~~~~~~~~


XY1200在XY1100的基础上对Modem的功耗/性能以及射频部分的集成度做了提升。
XY2100S在XY1200的基础上支持更多的功能，如LCD、AuxADC、运放、比较器等，专为智能表计、烟感行业高度优化定制。

特征参数
^^^^^^^^^^^

* 288 MHz :ref:`cortex_m4`
* 3x 5.33 Msps ADC
* sLib执行代码安全库区
* 16MB SPI Flash memory 扩展
* 4 x USART + 4 x UART
* 2 x CAN 2.0B
* Crystal-less USB2.0
* IEEE 1588 10/100M以太网MAC，专用DMA和4K SRAM，MII/RMII接口

    :ref:`at32f407` 基于 :ref:`cortex_m4` 的高性能产品，拥有丰富的通信外设资源，包括8个串口和以太网MAC，十分适用于网关类产品

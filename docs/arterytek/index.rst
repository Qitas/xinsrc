雅特力arterytek
=================

.. note::

    雅特力科技（重庆）有限公司于2016年6月成立，是一家致力于推动全球市场32位微控制器创新趋势的芯片(MCU)设计公司，2019年认定为高新技术企业。
    公司专注于ARM® Cortex®-M4 的32位微控制器研发与创新，分別在深圳、苏州、台湾都有设有研发、销售及技术支持分部。
    拥有领先高端芯片研发技术、完整的硅智财库及专业灵活的整合经验，创造了MCU 业界Cortex®-M4最高CPU主频288MHz运算效能，并致力于成为中国32位微控制器的创新领导者。

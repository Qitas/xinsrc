关键特性
~~~~~~~~~

* 工业级温度范围-40~105°C
* sLib执行代码安全库区
* Cortex®-M4主频最高 240 MHz
* 16MB SPI Flash memory 扩展
* 4 x USART + 4 x UART
* 2 x CAN 2.0B
* Crystal-less USB2.0
* IEEE 1588 10/100M以太网MAC，专用DMA和4K SRAM，MII/RMII接口

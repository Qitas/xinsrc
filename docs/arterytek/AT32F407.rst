Xin简介
-----------

:ref:`arterytek` :ref:`cortex_m4` 单核 MCU

关键特性
~~~~~~~~~

* 工业级温度范围-40~105°C
* sLib执行代码安全库区
* Cortex®-M4主频最高 240 MHz
* 高达 16MB 外部 SPI Flash memory 扩展
* 4 x USART + 4 x UART
* 2 x CAN 2.0B
* Crystal-less USB2.0全速接口
* IEEE 1588 10/100M以太网MAC，专用DMA和4K SRAM，MII/RMII接口

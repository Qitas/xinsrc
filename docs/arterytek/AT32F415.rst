规格参数
~~~~~~~~~~~

基本参数
^^^^^^^^^^^

* 供电电压：2.4V - 3.6V
* 工作温度：-40°C - 105°C
* 内核频率：150 MHz

特征参数
^^^^^^^^^^^

* 15ch ADC（12bit 2MSPS）
* 3 x USART + 2 x UART
* SDIO
* CAN 2.0B
* USB 2.0 OTG，内置1280 Byte专用SRAM
* 2 x I2C + 2 x SPI
* ERTC (1ppm的分辨率)
* 2 x COMP (轨到轨比较器)
* sLib安全库 (Security Library)

.. note::
    :ref:`arterytek` 自行开发的 :ref:`arterytek_slib` 可支持密码保护指定范围程序区，方案商烧录核心算法到此区域，提供给下游客户做二次开发，强化了产品本身的安全性、可靠性和二次开发的使用便利度。

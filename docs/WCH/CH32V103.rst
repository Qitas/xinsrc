关键特性
~~~~~~~~~~~~~~

* 提供10M以太网接口，内置PHY
* BLE 4.2，Zigbee兼容IEEE 802.15.4规范，单端RF接口无需外部电感
* 支持3.3V和2.5V电源，范围2.1V～3.6V, 内置DC/DC转换
* SPI x 2 + UART x 4 (16C550/6Mbps)
* USB H/D

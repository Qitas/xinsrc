关键特性
~~~~~~~~~~~

基本参数
^^^^^^^^^^^

* 供电电压：2.7V ~ 5.5V
* 工作温度：40°C to +85°C
* 处理性能：90 :ref:`DMIPS`
* RAM容量：20 KB
* Flash容量：128 KB

特征参数
^^^^^^^^^^^

* :ref:`cortex_m3` 72 MHz
* USB 2.0 x 2
* CAN 2.0B
* SPI(M/S) x 2 + UART x 3 + IIC x 2
* 12bit ADC + 12bit DAC

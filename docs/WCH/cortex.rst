WCH Cortex-M
======================

:ref:`wch` :ref:`cortex`

2012年起陆续引入ARM等32位内核，我们加入了网络、USB2.0等高速接口，并设计了高速DMA仲裁机制。 后期又推出CH32F103等通用MCU芯片，适用于软件兼容、硬件引脚兼容、接口更专业的成熟生态应用。

品牌对比
~~~~~~~~~

作为一颗更纯粹的MCU，主要卖点是 :ref:`cortex_m7` ，没有家族赋能和外设配置优势，可选择一些高主频的 :ref:`cortex_m4` 方案，或者其他定位高性能的强内核产品 :ref:`top`


型号对比
~~~~~~~~~

RT1010 相对 :ref:`rt1020` 去掉了以太网和CAN配置，同时相关配置的资源减半，拥有较小的LQFP80封装

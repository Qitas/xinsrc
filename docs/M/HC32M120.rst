关键特性
~~~~~~~~~~~~~~

* 12bit 1MSPS ADC支持Timer事件触发
* 2 x OPA，最大驱动电流30mA
* 2 x CMP，支持窗口比较功能
* -40~105 ℃，2.7-5.5V

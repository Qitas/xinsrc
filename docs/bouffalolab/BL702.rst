规格参数
~~~~~~~~~~~

无线子系统包含 2.4G 无线电，BLE5.0 和 802.15.4 基带/MAC 设计。

微控制器子系统包含一个低功耗的 32 位 RISC CPU，高速缓存和存储器。电源管理单元控制低功耗模式。此外，还支持各种安全性能。

外围接口包括 USB2.0，以太网，红外遥控器，SPI，UART，ISO 17987，I2C，I2S，PWM，QDEC，KeyScan，ADC，DAC，PIR 和 GPIO。

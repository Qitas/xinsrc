功耗参数
^^^^^^^^^^^

* 工作电压：1.71V ~ 3.8V
* 待机电流 <830nA（RTC、RAM 保持）
* 复位/关断电流 <150nA
* 无线电 Rx、Tx 功率为 0dBm 时，电流 <5.3mA
* 1s CONN 间隔：~6uA

通信能力
~~~~~~~~~~~

支持 :ref:`ble` 5.0 、Zigbee、专有2.4GHz

BLE
^^^^^^^^^^^
``BLE`` ``–96dBm`` ``+8dBm`` ``ZigBee`` ``6LoWPAN``

* TX 输出功率：-20dBm 至 +8dBm
* RX 灵敏度：1Mbps 时为 -96Bm

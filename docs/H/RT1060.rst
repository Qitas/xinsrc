品牌对比
~~~~~~~~~

STM32H7系列的主要竞争对手是NXP的i.MX RT系列

* STM32H7支持32位SDRAM，RT1050仅支持16位SDRAM
* STM32H7有JPEG硬解，RT的分辨率是1368 * 768，H7是1024*768
* RT带的两个USB都带高速PHY，H7系列无，得外置
* RT1060系列带双网口，H7系列仅一个

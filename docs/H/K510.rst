规格参数
~~~~~~~~~~~

K510支持丰富的网络模型算子，包括常见的CNN、RNN和各类向量计算和数据处理操作。
支持裸机、嵌入式RTOS和Linux，并且移植了 OpenCV Python Numpy。同时，K510支持TensorFlow、PyTorch、ONNX和TVM等主流深度学习框架，兼容性更强。

Xin
=====================

**新产品用心，芯产品用新**

`SoC·芯 <https://www.SoC.Xin>`_ 通过关联整理，多维度对比，求解如下问题：

* ``有新需求，评估实现途径，求参考值``
* ``有芯需求，技术选型对比，求最优解``
* ``有薪需求，知识领域更新，求解区间``
* ``有信需求，限定应用场景，归纳证明``

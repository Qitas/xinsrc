Nation
=================

`国民技术 <https://www.nationstech.com>`_ 股份有限公司于2000年源于国家“909”集成电路专项工程成立，2010年创业板上市（股票代码:300077），是通用MCU、安全芯片领先企业和国家高新技术企业，拥有博士后科研工作站。总部位于深圳，在北京、上海、武汉、西安、重庆、香港、新加坡、洛杉矶、日本等地设有分支机构。主营产品包括：通用MCU、安全芯片、可信计算芯片、智能卡芯片、非接读写芯片、蓝牙芯片、RCC创新产品等，广泛应用于网络安全认证、电子银行、电子证照、移动支付与移动安全、物联网、工业联网及工业控制、智能家电及智能家庭物联网终端、消费电子、电机驱动、电池及能源管理、智能表计、医疗电子、汽车电子、安防、生物识别、通讯、传感器、机器自动化等应用方向。

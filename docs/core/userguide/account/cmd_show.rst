 
.. _cmd_account_show:

pio account show
================

.. contents::

Usage
-----

.. code-block:: bash

    pio account show

Description
-----------

Show detailed information about :ref:`pioaccount`:

* Active subscriptions
* Available packages and services

Options
~~~~~~~

.. program:: pio account show

.. option::
    --json-output

Return the output in `JSON <http://en.wikipedia.org/wiki/JSON>`_ format

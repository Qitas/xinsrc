 
.. _cmd_team_destroy:

pio team destroy
================

.. versionadded:: 5.0

.. contents::

Usage
-----

.. code-block:: bash

    pio team destroy ORGNAME:TEAMNAME

Description
-----------

Destroy a team.

If you need to remove a user from a team, please use :ref:`cmd_team_remove` command.

测试说明：
编译并下载程序。

测试结果：
下载完程序后，完成是MCU通过SPI2对w5500的读写操作，完成对mac，gateway等的操作，内网测试，
请将学习板与路由器通过网线相接。并确认路由器的DHCP功能处于打开的状态。分配成功的话，会
通过串口1把网络参数打印到串口软件上（9600-n-8-1）

串口输出：
	源地网络适配板作为DHCP客户端，尝试从DHCP服务器获取IP地址 
	dhcp_state=2 IP from DHCP
	W5500 IP地址   : 192.168.1.167
	W5500 子网掩码 : 255.255.255.0
	W5500 网关     : 192.168.1.1
	已从DHCP服务器成功获得IP地址

硬件连接:
led:
	D1->PC13
	D2->PB0
	D3->PB1
USART1：
	PA9--MCU_TX    
	PA10---MCU_RX

W5500:(SPI2)
	SCLK---PB13
	MISO---PB14
	MOSI---PB15
	SCS----PB12

	W5500_INT---PB10
	W5500_RST---PB11
